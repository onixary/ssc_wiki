# 塑形盔甲

塑形盔甲在1.0.71版本后加入。它们是游戏中后期的盔甲，可以无视形态的盔甲相关限制，被绝大多数形态装备

你需要使用未加工的月尘，金锭，紫水晶碎片与对应的钻石护甲来合成塑形护甲

!!! note

	出于显而易见的原因，allay_sp无法装备塑形绑腿与塑形脚镯
